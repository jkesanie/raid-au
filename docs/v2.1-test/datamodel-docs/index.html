    Name: RAiD-core

    Add here some general information about the documentation site.

    Classes

    Class Description
    Access a metadata schema block containing RAiD access information
    AccessStatement a metadata schema block containing an explanation for any access type that is not ‘open’, with the explanation’s associated properties.
    AccessType None
    AlternateIdentifier a metadata schema block containing alternative local or global identifiers for the project or activity associated with the RAiD
    AlternateUrl a metadata schema block containing alternative URLs for the project
    ClosedRaid None
    Contributor a metadata schema block containing a contributor to a RAiD and their associated properties
    ContributorPosition a metadata schema sub-block describing a contributor’s administrative position on a project or activity
    ContributorRole None
    Date Metadata schema block containing the start and end date of the RAiD.
    Description a metadata schema block containing the description of the RAiD and associated properties
    DescriptionType a metadata schema block declaring the type of description
    Id a metadata schema block containing the RAiD name and associated properties.
    Language None
    Metadata None
    Organisation a metadata schema block containing the organisation associated with a RAiD and its associated properties
    OrganisationRole a metadata schema sub-block describing an organisation’s role on a project
    Owner None
    RaidDto None
            RaidCreateRequest None
            RaidUpdateRequest None
    RaidPatchRequest None
    RegistrationAgency metadata schema sub-block declaring the Registration Agency that minted the RAiD.
    RelatedObject a metadata schema block containing inputs, outputs, and process documents related to a RAiD plus associated properties
    RelatedObjectCategory a metadata schema sub-block declaring that a relatedObject is an input, output and/or process document
    RelatedObjectType a metadata schema sub-block describing a relatedObject’s type
    RelatedRaid a metadata schema block containing related RAiDs and qualifying the relationship
    RelatedRaidType a metadata schema sub-block qualifying the relationship with a related Raid
    SpatialCoverage metadata schema block containing information about any spatial region(s) or named place(s) targeted by the project
    SpatialCoveragePlace metadata schema sub-block containing free-text place names or descriptions plus associated metadata properties
    Subject metadata schema block containing the subject area of the RAiD plus associated properties
    SubjectKeyword metadata schema sub-block containing free-text keyword describing a project plus associated properties
    Title a metadata schema block containing the title of the RAiD and associated properties
    TitleType None
    TraditionalKnowledgeLabel metadata schema block containing information about Traditional Knowledge / Biocultural Labels and Notices
